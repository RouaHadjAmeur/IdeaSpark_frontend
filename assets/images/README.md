# Assets Images

## Google Calendar Icon

Pour ajouter l'icône Google Calendar :

1. Téléchargez l'icône officielle depuis :
   - https://developers.google.com/identity/branding-guidelines
   - Ou utilisez une icône de calendrier générique

2. Nommez le fichier : `google_calendar_icon.png`

3. Placez-le dans ce dossier : `assets/images/`

4. Dimensions recommandées : 512x512 px (PNG avec transparence)

## Alternative

Si vous n'avez pas l'icône, le code utilise automatiquement l'icône Material `Icons.calendar_today` comme fallback.
