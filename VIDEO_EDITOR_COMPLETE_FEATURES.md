## 🎯 **UTILISATION**

### **Ajouter du Texte**
1. Appuyer sur l'icône **Texte** (T)
2. Saisir le texte dans le dialogue
3. Ajuster la taille, gras, italique
4. **Pincer pour zoomer** sur le texte ajouté
5. **Glisser pour déplacer**
6. **Toucher pour éditer** (couleurs, emojis, styles)

### **Appliquer des Filtres**
1. Sélectionner l'onglet **Filtres** (par défaut)
2. Faire défiler horizontalement les aperçus
3. Toucher un filtre pour l'appliquer instantanément

### **Dessiner**
1. Appuyer sur l'icône **Pinceau**
2. Choisir une couleur via l'onglet **Couleurs**
3. Dessiner directement sur la vidéo
4. Utiliser le bouton de nettoyage pour effacer

### **Sauvegarder**
1. Appuyer sur l'icône **Téléchargement** en haut à droite
2. La vidéo se met en pause automatiquement
3. Capture complète avec tous les overlays
4. Sauvegarde dans "Vidéos Éditées"
